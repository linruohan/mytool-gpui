# 键盘快捷键使用指南

## 概述

MyTool 提供了完整的键盘快捷键系统，让您无需鼠标即可高效完成大部分操作。

> **注意**: 文档中的 `Cmd` 在 macOS 上对应 Command 键，在 Windows/Linux 上对应 Ctrl 键。

## 快捷键列表

### 任务操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Cmd + N` | 新建任务 | 在当前视图创建新任务 |
| `Cmd + E` | 编辑任务 | 编辑选中的任务 |
| `Cmd + D` | 删除任务 | 删除选中的任务 |
| `Cmd + Enter` | 完成/取消完成 | 切换任务的完成状态 |
| `Cmd + P` | 置顶/取消置顶 | 切换任务的置顶状态 |
| `Cmd + Shift + D` | 复制任务 | 复制选中的任务 |
| `Cmd + L` | 添加标签 | 为任务添加标签 |
| `Cmd + T` | 设置截止日期 | 设置任务的截止日期 |

### 导航

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Cmd + 1` | 收件箱 | 显示收件箱视图 |
| `Cmd + 2` | 今日任务 | 显示今日任务视图 |
| `Cmd + 3` | 计划任务 | 显示计划任务视图 |
| `Cmd + 4` | 已完成 | 显示已完成任务视图 |
| `Cmd + 5` | 置顶任务 | 显示置顶任务视图 |
| `Cmd + 6` | 标签视图 | 显示标签视图 |
| `Cmd + [` | 上一个视图 | 切换到上一个视图 |
| `Cmd + ]` | 下一个视图 | 切换到下一个视图 |

### 搜索和过滤

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Cmd + F` | 搜索任务 | 打开搜索框 |
| `Cmd + Shift + L` | 按标签过滤 | 打开标签过滤器 |
| `Cmd + Shift + P` | 按项目过滤 | 打开项目过滤器 |
| `Cmd + Shift + C` | 清除过滤器 | 清除所有过滤条件 |
| `Cmd + Shift + A` | 显示所有任务 | 显示所有任务（包括已完成） |

### 选择和批量操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Cmd + A` | 选择所有 | 选择当前视图的所有任务 |
| `Esc` | 取消选择 | 取消所有选择 |
| `↑` | 上一个任务 | 选择上一个任务 |
| `↓` | 下一个任务 | 选择下一个任务 |
| `Cmd + Shift + Enter` | 批量完成 | 完成所有选中的任务 |
| `Cmd + Shift + Delete` | 批量删除 | 删除所有选中的任务 |
| `Cmd + Shift + M` | 批量移动 | 移动选中的任务到其他项目 |

### 项目和分区

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Cmd + Shift + N` | 新建项目 | 创建新项目 |
| `Cmd + Shift + E` | 编辑项目 | 编辑当前项目 |
| `Cmd + Shift + D` | 删除项目 | 删除当前项目 |
| `Cmd + Shift + A` | 归档项目 | 归档当前项目 |
| `Cmd + Alt + N` | 新建分区 | 在当前项目创建新分区 |
| `Cmd + Alt + E` | 编辑分区 | 编辑当前分区 |
| `Cmd + Alt + D` | 删除分区 | 删除当前分区 |

### 视图和窗口

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Cmd + B` | 切换侧边栏 | 显示/隐藏侧边栏 |
| `Cmd + Shift + F` | 切换全屏 | 进入/退出全屏模式 |
| `Cmd + =` | 放大 | 放大界面 |
| `Cmd + -` | 缩小 | 缩小界面 |
| `Cmd + 0` | 重置缩放 | 重置界面缩放到 100% |
| `Cmd + R` | 刷新视图 | 刷新当前视图 |
| `Cmd + ,` | 打开设置 | 打开应用设置 |
| `F1` | 打开帮助 | 打开帮助文档 |

## 使用技巧

### 1. 快速添加任务

```
1. 按 Cmd + N 打开新任务对话框
2. 输入任务内容
3. 按 Enter 保存
```

### 2. 批量处理任务

```
1. 按 Cmd + A 选择所有任务
2. 按 Cmd + Shift + Enter 批量完成
```

### 3. 快速导航

```
1. 按 Cmd + 1 切换到收件箱
2. 按 Cmd + 2 切换到今日任务
3. 按 Cmd + 3 切换到计划任务
```

### 4. 搜索和过滤

```
1. 按 Cmd + F 打开搜索
2. 输入关键词
3. 按 Enter 搜索
4. 按 Cmd + Shift + C 清除过滤器
```

### 5. 键盘导航

```
1. 按 ↓ 选择下一个任务
2. 按 ↑ 选择上一个任务
3. 按 Enter 编辑选中的任务
4. 按 Cmd + Enter 完成选中的任务
```

## 自定义快捷键

目前快捷键是预定义的，未来版本将支持自定义快捷键配置。

## 快捷键冲突

如果快捷键与系统或其他应用冲突，可以：

1. 在系统设置中禁用冲突的快捷键
2. 等待未来版本支持自定义快捷键

## 常见问题

### Q: 快捷键不工作？

A: 请确保：
- 应用窗口处于焦点状态
- 没有其他对话框或弹窗打开
- 快捷键没有被系统占用

### Q: 如何查看所有快捷键？

A: 按 `F1` 打开帮助，或查看本文档。

### Q: 可以自定义快捷键吗？

A: 当前版本不支持，未来版本将添加此功能。

### Q: macOS 和 Windows 的快捷键有区别吗？

A: 主要区别是修饰键：
- macOS: Command (⌘)
- Windows/Linux: Ctrl

其他按键保持一致。

## 快捷键设计原则

1. **一致性**: 遵循平台标准（Cmd/Ctrl + N 新建，Cmd/Ctrl + S 保存等）
2. **可发现性**: 在菜单和工具提示中显示快捷键
3. **避免冲突**: 不与系统快捷键冲突
4. **分组逻辑**: 相关功能使用相似的快捷键组合
5. **易记性**: 使用助记符（如 Cmd + L 添加 Label）
